# 🇪🇸 Spain Beckham Law Tax Calculator (2026 Edition)

A high-precision, single-page application (SPA) to help expats, digital nomads, and high-net-worth individuals determine if they should opt for the **Special Tax Regime for Displaced Workers** (the "Beckham Law").

## 🚀 Why this exists
Spain's tax system is highly progressive, reaching up to 47% (or higher in regions like Catalonia). The **Beckham Law** allows you to pay a flat **24%** on Spanish-source income. However, because you lose personal allowances and social security deductions, it isn't always the cheapest option for salaries under €58,000.

## 🛠 Features
- **2026 Tax Brackets:** Includes the latest state and average regional IRPF rates.
- **Social Security Logic:** Accounts for the 2026 MEI (Intergenerational Solidarity Mechanism) and the capped contribution bases.
- **Foreign Income Exemption:** Visualizes the massive savings for those with global dividends or rental income.
- **Visual Charts:** Instant comparison using Chart.js.

## 📈 The Break-Even Point
In 2026, the general rule of thumb is:
- **Salary < €55k:** Standard Regime is usually better.
- **Salary > €60k:** Beckham Law is almost always better.
- **Foreign Income:** If you have > €5k in foreign dividends, Beckham Law is better even at lower salary levels.
